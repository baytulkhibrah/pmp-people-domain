## 🚀 طريقة النشر على GitHub Pages

### الطريقة الأولى: عبر واجهة GitHub (الأسهل)

1. أنشئ مستودعاً جديداً على GitHub (مثلاً: `pmp-people-domain`).
2. ارفع الملفات التالية إلى المستودع:
   - `index.html`
   - `questions-data.js`
   - `README.md` (هذا الملف)
3. اذهب إلى **Settings → Pages**.
4. تحت **Source** اختر **Deploy from a branch**.
5. اختر الفرع `main` والمجلد `/ (root)` ثم اضغط **Save**.
6. انتظر دقيقة أو دقيقتين، وسيصبح موقعك متاحاً على:
   ```
   https://<اسم-المستخدم>.github.io/pmp-people-domain/
   ```

### الطريقة الثانية: عبر سطر الأوامر

```bash
git init
git add index.html questions-data.js README.md
git commit -m "Initial commit: PMP People Domain question bank"
git branch -M main
git remote add origin https://github.com/<your-username>/pmp-people-domain.git
git push -u origin main
```

ثم فعّل GitHub Pages من **Settings → Pages** كما في الخطوات أعلاه.

## 🎮 التشغيل المحلي

ما عليك إلا فتح `index.html` في أي متصفح. لا حاجة لخادم.

أو إذا أردت تشغيل خادم محلي بسيط:

```bash
# Python 3
python3 -m http.server 8000
# ثم افتح: http://localhost:8000
```
